# 🔥🤖 Smart Fire Fighting Robot using Arduino

## 📌 Project Overview

This project is an **Arduino-based Smart Fire Fighting Robot** that can automatically detect fire using IR flame sensors and extinguish it using a water pump controlled by a servo motor. The robot also uses an L298N motor driver for movement and can navigate toward the fire source.

---

## 🚀 Features

* 🔥 Automatic fire detection using IR sensors
* 🚗 Autonomous movement using DC motors
* 💧 Water spraying system using pump and servo motor
* 🔄 Directional fire targeting (Left, Right, Front)
* 📡 Real-time sensor monitoring via Serial Monitor

---

## 🧠 Working Principle

1. IR flame sensors detect fire intensity from three directions (Left, Front, Right).
2. Robot analyzes sensor values and moves toward the fire.
3. Once fire is detected:

   * Robot stops
   * Water pump is activated
   * Servo motor rotates to spray water in multiple directions
4. If no fire is detected, robot continues searching.

---

## 🧩 Components Used

* Arduino UNO / Nano
* L298N Motor Driver Module
* 2x DC Motors
* IR Flame Sensors (3x)
* Servo Motor (SG90)
* Mini Water Pump
* Relay Module (optional)
* Battery Pack (7.4V–12V)
* Chassis (Robot body frame)

---

## 🛠️ How to Run

1. Assemble the robot hardware

3. Power the system using battery
4. Place in test environment
